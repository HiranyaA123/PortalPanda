Install in Gmail
  1. Click Copy signature.
  2. Open Gmail Settings, then See all settings > General > Signature.
  3. Create a new signature, click inside its editor, and paste.
  4. Choose it under Signature defaults, then save your changes.

The logo loads from centralpass.au, so deploy the latest site version before installing the signature. This ensures recipients can load the image from a public HTTPS URL.
